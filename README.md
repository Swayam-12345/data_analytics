# Sales Performance Dashboard

## Description
Designed and developed an interactive dashboard to track and analyze key sales KPIs, including revenue, profit, and growth trends. Implemented data visualization techniques to present region-wise and product-wise insights, helping in identifying high-performing areas and improvement opportunities. Built with a focus on usability and clarity to support effective business decision-making.


## 🚀 Features
- Track total sales, revenue, and profit
- Region-wise and product-wise analysis
- Monthly and yearly sales trends
- Interactive filters for dynamic insights
- User-friendly and clean interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📈 Key Insights
- Identified top-performing products and regions
- Analyzed seasonal sales trends
- Highlighted areas with low performance for improvement
